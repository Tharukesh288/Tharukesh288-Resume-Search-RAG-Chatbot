# Resume Search Chatbot (RAG + Gemini)

A Streamlit-based resume search chatbot using:
- LangChain
- ChromaDB
- HuggingFace embeddings
- Google Gemini

## Features
- Upload multiple resume PDFs
- Semantic search by skills or role
- Gemini-powered answers
